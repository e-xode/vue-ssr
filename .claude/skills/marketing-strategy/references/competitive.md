# Competitive landscape — Vue SSR Starter Kit

The competitor map and the differentiation watch list. Keep claims factual; do not denigrate competitors
in produced content.

## Category

Vue/JavaScript SSR starter kits and boilerplates. Adjacent categories: meta-frameworks (Nuxt),
general full-stack templates, and paid SaaS-starter boilerplates.

## Competitor map

| Type | Examples | How the kit differs |
| --- | --- | --- |
| Vue meta-frameworks | Nuxt | The kit is a transparent, owned Express 5 + Vite SSR stack you fully control, not a framework abstraction; ships auth/admin/i18n already wired |
| Vue SSR boilerplates | Generic GitHub Vue+SSR templates | Production defaults (Helmet, rate limiting, CI, Docker, tests) and the curated Claude Code config, not a bare scaffold |
| Paid SaaS-starter kits | Commercial Vue/JS boilerplates | Free and MIT, fully readable, no paywall or lock-in |
| AI-config templates | Repos shipping AI rules only | A complete production app plus the AI fleet, not just a config file |

(Confirm specific competitor feature claims before publishing; this map is for internal positioning.)

## Differentiation spine

- **AI-pair-programming-ready** — the curated Claude Code fleet/skills/rules/hooks ship with the kit.
- **Production-grade, not a toy** — auth, security, i18n, admin, CI, Docker, tests included.
- **Free and MIT** — full source, no paywall, no lock-in.
- **Owned, transparent stack** — Vue 3 + Express 5 + MongoDB you control, not a framework black box.

## Watch list

- Nuxt and other meta-frameworks improving their out-of-the-box auth/admin story.
- New AI-config-bundled starter kits emerging as "AI-ready" becomes table stakes.
- Vue / Vite / Vuetify / Express major releases that shift the stack story (re-read `package.json`).
- Paid boilerplates that go free, or free ones that add the AI-config angle.
